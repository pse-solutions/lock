Understanding the Gateway Menu (Mobile App)

Gateway List Overview

Once you`ve tapped Gateway in the main menu, you`ll see each gateway entry organized into columns. Here`s what each represents:

  • Gateway Type: Displays the model name (G2, G3, G3P, G4).
  • Gateway Status:
    • Online indicates the gateway is connected to the network.
    • Offline indicates no network connection or power.
  • Connected Locks: Number of locks currently paired to this gateway.
