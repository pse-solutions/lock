Access Gateway Settings in the TTLock Web System

1. Log In to the Web Portal

1. Open your desktop browser and navigate to https://web.ttlock.com.
2. Sign in with your registered TTLock account credentials.

2. Click “Gateways” in the Top Menu

Once logged in, locate the top navigation bar and click Gateways. This will load the Gateway List page.

3. Gateway List Overview (Web)

The web view displays each gateway in a table, including:

  • Gateway Name and Lock Model
  • Battery Power
  • Connection Status
