View Gateway ↔ Lock Connection Strength (Mobile App)

1. Navigate to “Nearby Locks”

From the Gateway Detailed View, tap the Nearby Locks button to display all locks currently within the gateway’s radio range.

2. Interpreting Connection Strength Indicators

A list of locks appears, each with a signal‐strength indicator:

  • Strong: Optimal signal strength (ideally positioned).
  • Medium: Sufficient but may occasionally drop (consider relocating).
  • Weak: Unreliable connection (move lock or gateway closer).

3. Multiple Gateway Connections

Each lock can be paired with more than one gateway. If a lock shows a “Weak” connection here, check its signal on other gateways (stronger links may exist). Always choose the gateway showing a “Strong” connection for mission-critical doors.

4. Improving a Weak Connection

  • Relocate the lock or gateway to minimize obstacles (e.g., metal doors, thick walls).
  • Consider adding a Wi-Fi repeater or another gateway to create mesh coverage.
  • Ensure the gateway`s firmware is up to date (check “Update Firmware” in the Detailed View).
