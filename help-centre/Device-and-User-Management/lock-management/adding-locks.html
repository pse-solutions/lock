Adding Locks

TTLock iOS or Android App only

Must be next to the hardware with Bluetooth enabled on your mobile device

A lock must be added to an account before it can be used.

You cannot add a lock that is already associated with another account, if the lock you are trying to add is associated with another account, you must either:

  • Delete the lock from that account
  • Transfer the lock from that account
  • Reset the lock

Steps to add a lock

  1. Tap the Menu icon on the main lock page, tap "+ Add Lock" and then tap "Door Lock".
  1. Press any button on the lock keypad to activate pairing mode. You'll hear "Please add a Bluetooth administrator." Select the lock with the "+" icon from the list of nearby locks that appears.

New locks will appear in blue, locks that have already been added to an account will appear in grey.

  1. Assign a name to the lock. For certain lock types, you'll also need to specify the door's opening direction.
