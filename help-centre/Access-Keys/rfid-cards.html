Cards (RFID Keys)

This requires a registered TTLock account

  • Download the TTLock application on Android or iOS devices
  • Access the TTLock Web System on desktop here

Overview

Cards (aka RFID keys) are a more traditional type of access method, with the majority of properties already leveraging this technology.

Adding cards

1. Add a card using a lock

Must be next to the hardware with Bluetooth enabled on your mobile device

TTLock iOS or Android App only

Any lock can be used to add a card into the system. Once the card is added, it can then be assigned to other locks using the TTLock Web Application.

  1. From the home screen, select a lock
  2. Tap "Cards"
  3. Choose if you want the card to be "Permanent", "Timed", or "Recurring"
    1. If "Timed" or "Recurring", set the validity period
  4. Assign a name and tap next
  5. The lock will ask you to present the card
  6. Present the card against the front of the lock

If at this point you want to program this card onto other locks too, follow these steps:

  1. Log into the TTLock Web Application
  2. Click "Issue Card"
  3. Click "Sync Card" and the card you added to the lock will appear in the list
  4. Click "Issue" next to the card you want to program and then choose the locks that you want that card to be programmed to and click "OK"

2. Add a card using a card reader

TTLock Web Console only

Lock must be connected to a gateway

The card encoder must be plugged into a laptop or desktop computer, and the user must be logged into the TTLock Web Console.

  1. From the home screen, click "Issue Card"
  2. Choose "Issue Card" from the menu bar
  3. Select one or more locks you want the card to be programmed to
  4. Name the card (it's recommended to use an easy naming convention)
  5. Choose if you want the card to be "Permanent", "Timed", or "Recurring"
    1. If "Timed" or "Recurring", set the validity period
  6. Click "Read Card"
  7. Tap the card on the card reader

Use the "Search" function to quickly identify any card registered in the system by tapping the card against the card reader.

Delete a card

Lock must be connected to a gateway

  1. From the home screen, click "Issue Card"
  2. Locate the card you wish to delete, then click "Door access"
  3. Select the locks from which you want to remove access
    1. Select the top box to remove access from all locks
  4. Click "Delete access", then "OK"
  5. Confirm that the access has been removed from each lock
  6. Click "OK" to complete
