Passcodes (PIN Codes)

This requires a registered TTLock account

  • Download the TTLock application on Android or iOS devices
  • Access the TTLock Web System on desktop here

Overview

Passcodes (aka PIN Codes) are a convenient and key-free way for different users to access a building. Often considered the most flexible credential, adapting well to any application. PIN codes do not require a medium, such as a card or device.

Unsure of the best way to manage passcodes at your property? Check out this blog post for more insights.

Passcode Types

1. Online codes

Includes: Custom

  • Requires either a connection to a gateway or a direct Bluetooth connection to the lock.
  • They can be added, modified, or deleted in real-time remotely, or directly at the lock.
  • They can be customized to any 4 to 9 digit code.

Custom

  • Choose whether this code should work permanently or set a specific start and end time
  • Set specific start and end times (accurate to the hour)

2. Offline codes

Includes: Timed, One-Time, Permanent, Recurring, Erase

  • These are generated algorithmically and don't require a Bluetooth connection to the lock or a network gateway.
  • Although they can be generated without a connection, they do require a Bluetooth connection to the lock to be modified or deleted.
  • They must be used within 24 hours (except for one-time codes, which must be used within 6 hours) or they will be suspended for security reasons.
  • Offline codes are randomly generated and range from 6 to 9 digits in length.

Timed

  • Set specific start and end times (accurate to the hour)

One-time

  • The code is valid for a single use only and becomes inactive afterwards

Permanent

  • The code remains valid and can be used indefinitely until it's deleted

Recurring

  • Choose a recurring cycle for the code: daily, weekend, workday, or a specific day of the week
  • Set specific start and end times (accurate to the hour)
  • The code is only valid during the specified cycle

Erase

  • Generate this code to remove all codes from a lock
  • Once entered into the lock, all existing codes will be permanently erased

Adding Passcodes

Steps to generate passcodes in mobile app

  1. Select a lock from the main menu, then tap "Passcodes", then tap "Generate Passcode".
  1. Choose one of the Passcode types (Timed, Custom, etc.), and If applicable, specify the start and end times and assign a name, then tap "Generate". Once the passcode is generated, it can optionally be shared via email or SMS directly from the app using a pre-set template.

Steps to generate passcode in web system

  1. From the main lock menu, click the number pad icon on the lock you want to generate a passcode for.
  1. Click the "+" button to select additional locks to add the passcode to, enter a "Name" and select one of the Passcode types (Timed, Custom, etc.). If applicable, specify the start and end times and assign a name, then click "OK". Once the passcode is generated, it can optionally be shared via email or SMS directly from the app using a pre-set template.
