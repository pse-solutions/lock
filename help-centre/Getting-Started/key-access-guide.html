Creating Keys

The TTLock platform supports various access methods:

  1. Passcodes (PIN Codes)
  2. Remote Unlock (WiFi Open)
  3. Cards (RFID Keys)
  4. eKeys (Digital Keys)
  5. Fingerprints

Passcodes (PIN Codes)

Passcodes (aka PIN Codes) are a convenient and key-free way for different users to access a building. They are considered the most flexible credential, adapting well to any application, and do not require a medium, such as a card or device.

Passcode Types

  • Online codes:

    Online codes require either a connection to a gateway or a direct Bluetooth connection to the lock.

    They can be added, modified, or deleted in real-time remotely, or directly at the lock.

  • Offline codes

    Offline codes are generated algorithmically and do not require a Bluetooth connection to the lock or a network gateway.

    Although they can be generated without a connection, they do require a Bluetooth connection to the lock to be modified or deleted.

    They must be used within 24 hours (except for one-time codes, which must be used within 6 hours) or they will be suspended for security reasons.

Remote Unlock (WiFi Open)

This feature allows users to unlock doors from any location, without needing to be near the lock.

It requires the lock to be connected to a gateway. This function can be performed from anywhere, so it's a good idea to consider the security implications before proceeding.

If you're using a third-party automation partner integrated with TTLock, remote unlock must be enabled.

Cards (RFID Keys)

Cards (aka RFID keys) are a more traditional type of access method, with the majority of properties already leveraging this technology.

Cards can be added into the system using a lock or by using a card reader.

Any lock can be used to add a card into the system. Once the card is added, it can then be assigned to other locks using the TTLock Web Application.

eKeys (Digital Keys)

eKeys are digital keys. Types of eKeys include: Timed, One-Time, Permanent, or Recurring.

eKeys can be sent to users via email or SMS, and they can be used with the TTLock app on mobile devices.

Once users are granted access to an eKey, they will be prompted to create a TTLock account and access the key from their mobile devices.

eKey Types

  • Bluetooth eKeys: use BLE technology which requires users to be in close proximity to the lock they want to open.
  • Remote Unlock eKeys: allow users to open door locks from their mobile device using the internet. With Remote Unlock eKeys, users can open the doors from anywhere, so long as their mobile device has internet connectivity.

Fingerprints

Fingerprint access allows for quick and secure entry without the need for physical keys or cards.

Fingerprint access can be granted to staff or long-term residents and must be programmed at each individual lock.
