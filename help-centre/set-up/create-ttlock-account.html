Create a TTLock Account

This free account lets you manage all locks and gateways.

Step-by-step (mobile)

  1. Install the TTLock app (iOS | Android).
  2. Tap Register.
  3. Choose Email or Phone and enter the code sent to you.
  4. Accept the Terms & Privacy Policy and tap Register.
  5. Log in with the same email/phone on any device or via the TTLock web console.

Already have an installer account? Use it – you don’t need a separate owner login.
