eKeys (Digital Keys)

  • Download the TTLock application on Android or iOS devices
  • Access the TTLock Web System on desktop here

Overview

Bluetooth eKeys

Bluetooth eKeys use BLE technology which requires users to be in close proximity to the lock they want to open.

In hospitality, it's most common for guests to be granted Bluetooth eKeys.

Remote Unlock eKeys

Remote Unlock eKeys allow users to open door locks from their mobile device using the internet. With Remote Unlock eKeys, users can open the doors from anywhere, so long as their mobile device has internet connectivity.

Adding eKeys

Steps to generate eKeys using app

Once users are granted access to an eKey, they will be prompted to create a TTLock account and access the key from their mobile devices.

  1. Select a lock from the main menu, then tap "eKeys", then tap "Send eKey".
  1. Choose from Timed, One-Time, Permanent, or Recurring eKey types and enter the required information.

At this step, you can choose whether to also grant the user a "Remote Unlocking" key.

  1. Once the eKey is generated, it can optionally be shared via email or directly from the app using a pre-set template.

Steps to generate eKeys using web system

This is a paid feature

  1. Select a lock from the main menu, then tap "eKeys", then tap "Send eKey".
  1. Choose from Timed, One-Time, Permanent, or Recurring eKey types and enter the required information and tap "OK".

Here, you can also select whether to grant the user "Remote Unlock" and /or "Authorized Administrator" permissions to the lock.

  1. Tap "Send Email" to send the eKey to the recipient.
