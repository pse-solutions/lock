Pair a Gateway

Gateways bring your locks online so you can unlock and change settings from anywhere.

Network requirements

  • G2 Wi-Fi – 2.4 GHz SSID.
  • G3 / G3P (PoE) – open firewall port 2999.
  • G4 (4G) – nano-SIM with data plan.

Add the gateway (mobile)

  1. In the TTLock app, tap Menu → Gateway → +.
  2. Select the correct model and power it up until the LED flashes red/blue.
  3. Tap the + next to the gateway once it appears.
    (For G2, choose the Wi-Fi network and enter the password.)
  4. The gateway auto-connects to nearby locks; check signal strength to confirm.

One lock can talk to multiple gateways – ideal for large corridors.

Troubleshooting

If the gateway stays offline for more than 60 s, unplug and plug it back in, then repeat the add process.
