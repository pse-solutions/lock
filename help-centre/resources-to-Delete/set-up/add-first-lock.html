Add Your First Lock

You must be next to the lock with Bluetooth turned on.

Before you begin

  • The lock is installed and has fresh batteries.
  • The lock is not already bound to another account. If it is, delete, transfer, or factory-reset it first.

Steps (TTLock mobile app)

  1. Open the app ➜ tap the + ➜ choose Add Lock → Door Lock.
  2. Wake the lock keypad (it announces “Please add a Bluetooth administrator”).
  3. Select the blue + lock in the list.
  4. Name the lock and set the door-opening direction if prompted.

Need to move ownership later? See lock transfer instructions.
