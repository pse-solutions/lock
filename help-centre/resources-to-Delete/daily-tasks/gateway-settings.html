Gateway Settings

Rename gateways, check status, and view lock signal strength.

Before you begin

  • TTLock account with gateways added.
  • Mobile app or TTLock web console.

Step-by-step (mobile)

  1. Open Menu → Gateway to list all gateways.
  2. Tap a gateway to enter the detailed view.
  3. Rename, view MAC, or update firmware as needed.
  4. Tap Nearby Locks to see each connected lock and its signal (Strong / Weak).

What could go wrong?

Gateway offline

Check power, Wi-Fi 2.4 GHz or PoE port 2999, then reboot.

Lock shows weak signal

Remember a lock can talk to multiple gateways—pair another nearby unit for redundancy.
