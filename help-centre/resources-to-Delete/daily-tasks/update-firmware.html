Update Firmware

Install the latest firmware to unlock new features and security patches.

Before you begin

  • Phone beside the lock with Bluetooth ON.
  • Battery ≥ 30 % (check in the lock list).

Step-by-step

  1. Lock → Settings → Firmware Update → Check for UPDATE. :contentReference[oaicite:16]{index=16}:contentReference[oaicite:17]{index=17}
  2. If a version appears in blue, tap Update.
  3. Keep the app foregrounded; update finishes in 60 s.
  4. “Success” displays — the lock restarts automatically.

What could go wrong?

Update fails or stalls

Move closer to the lock, disable battery-saver, and retry. :contentReference[oaicite:18]{index=18}:contentReference[oaicite:19]{index=19}
