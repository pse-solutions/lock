Manage Administrators

Transfer ownership or add authorised administrators to selected locks.

Before you begin

  • You are the current lock administrator.
  • Recipient accounts are already registered on TTLock.

Step-by-step

  1. Transfer administrator – Settings ▶ Transfer Lock(s); choose locks, enter recipient, confirm.
  2. Add authorised admin (web) – click eKey ▶ +, select locks, input recipient, toggle Authorized Administrator.
  3. Add via mobile – lock ▶ Authorized Admin ▶ enter email/phone ▶ Send.

What could go wrong?

Recipient not found

Have them register first using the Account Registration guide.

Cannot see Authorized Admin icon

You are not the lock administrator—check permissions on the main lock list.
