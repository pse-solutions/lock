Complete Setup Guide Overview for 33 Lock System

Step 1: Account Registration

  • Download the TTLock application on Android or iOS devices.
  • Register and create account using phone or email
  • Verify account with code sent via text/email & accept user terms and services

Step 2: Physical Lock Installation

  • Follow installation manual for your lock model (IntelliMortise, IntelliLever, IntelliBolt, etc.)
  • Install mortise/lock chassis into door
  • Connect front and back lock components with cable
  • Install strike plate on door frame
  • Insert batteries (4x AAA for IntelliMortise, 3x AA for other models)
  • Verify lock powers up when batteries inserted

Step 3: Add Locks to TTLock App

  • Ensure Bluetooth is enabled on your mobile device
  • Stand next to lock with Bluetooth enabled on your mobile device
  • Tap Menu → "+ Add Lock" → "Door Lock"
  • Press any button on lock keypad to activate pairing
  • Select lock from list (appears in blue)
  • Name the lock and specify door opening direction

See detailed Account Setup Guide here

Note: If the lock you are trying to add is already associated with another account, you must either delete or transfer the lock from that account

Step 4: Initialize Lock Settings (Required)

  • Update firmware to latest version
  • Turn on "Remote Unlock" feature
  • Adjust lock volume settings
  • Calibrate lock time
  • Configure auto-lock settings if applicable

Step 5: Add Gateways (Only for Remote Access)

  • Place gateways within 20-30 feet of locks
  • Connect gateway to power source
  • In app: Menu → "Gateway" → "+"
  • Wait for LED to flash red and blue
  • Select gateway model and connect
  • For G2: Enter WiFi credentials (2.4GHz only)
  • Allow 3-5 minutes for locks to auto-connect

Step 6: Set Up Administrators (If Needed)

  • Add authorized administrators for property management
  • Transfer lock ownership if installed by contractor
  • Grant specific permissions as needed

Step 7: Create Access Credentials

Passcodes (PIN Codes):

  • Generate custom 4-9 digit codes
  • Set as permanent, timed, one-time, or recurring

Remote Unlock:

  • Enable for instant access from anywhere
  • Requires gateway connection

Cards (RFID Keys):

  • Add cards directly at lock or via card reader
  • Assign cards to multiple locks through web console

eKeys (Digital Keys):

  • Send Bluetooth or remote unlock keys via email
  • Recipients must create TTLock account

Fingerprints:

  • Add at each individual lock
  • Scan fingerprint 4 times for enrollment

Step 8: Ongoing Maintenance

  • Change batteries when notification shows 20 percent remaining
  • Use backup power port if batteries die completely
  • Check gateway connections regularly
  • Update firmware when available (approximately 2x per year)
  • Monitor lock activity through app or web console

Important Notes:

  • Keep physical keys as backup
  • Gateways create mesh network for redundancy
  • All credentials work offline except remote features
  • Web console available for bulk management at TTLock desktop site
