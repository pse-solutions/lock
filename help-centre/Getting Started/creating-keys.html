Creating Keys

Step 5 of the Getting Started Guide is to Create Keys. The TTLock platform supports various access methods.

Passcodes (PIN Codes)

Passcodes (aka PIN Codes) are a convenient and key-free way for different users to access a building. They are considered the most flexible credential, adapting well to any application, and do not require a medium, such as a card or device.

Passcode Types

  1. Online codes
    • Includes: Custom.
    • Requires either a connection to a gateway or a direct Bluetooth connection to the lock.
    • They can be added, modified, or deleted in real-time remotely, or directly at the lock.
    • They can be customized to any 4 to 9 digit code.
    • Custom codes allow you to choose whether they work permanently or set a specific start and end time (accurate to the hour).
  2. Offline codes
    • Includes: Timed, One-Time, Permanent, Recurring, Erase.
    • These are generated algorithmically and don't require a Bluetooth connection to the lock or a network gateway to generate.
    • They do require a Bluetooth connection to the lock to be modified or deleted.
    • They must be used within 24 hours (except one-time, which is 6 hours) or they will be suspended.
    • Offline codes are randomly generated and range from 6 to 9 digits in length.
    • Types include:
      • Timed: Set specific start and end times (accurate to the hour).
      • One-time: Valid for a single use only.
      • Permanent: Remains valid indefinitely until deleted.
      • Recurring: Choose a recurring cycle (daily, weekend, workday, specific day of the week) and set start/end times; only valid during the specified cycle.
      • Erase: Generate this code to remove all codes from a lock; all existing codes are permanently erased once entered.

Steps to generate passcodes in mobile app

  1. Select a lock from the main menu, then tap "Passcodes", then tap "Generate Passcode".
  2. Choose one of the Passcode types (Timed, Custom, etc.), and if applicable, specify the start and end times and assign a name, then tap "Generate".

Once generated, the passcode can optionally be shared via email or SMS directly from the app using a pre-set template.

Steps to generate passcode in web system

  1. From the main lock menu, click the number pad icon on the lock you want to generate a passcode for.
  2. Click the "+" button to select additional locks to add the passcode to, enter a "Name" and select one of the Passcode types. If applicable, specify the start and end times and assign a name, then click "OK".

Once the passcode is generated, it can optionally be shared via email or SMS directly from the app using a pre-set template.

Remote Unlock (WiFi Open)

This feature allows users to unlock doors from any location, without needing to be near the lock. It requires the lock to be connected to a gateway. This function can be performed from anywhere, so it's a good idea to consider the security implications before proceeding. If you're using a third-party automation partner integrated with TTLock, remote unlock must be enabled.

Steps to turn on remote unlock

To enable remote unlock, it must be activated in the settings for each lock.

  1. Determine if remote unlock is already on. If it is, you will see the words "Remote Unlock" in grey directly under the lock name, or you can see the blue network icon once a lock is selected.
  2. If remote unlock is not already turned on, select the lock that you want to enable remote unlock, then tap on "Settings".
  3. Tap on "Remote Unlock", then Tap the button to "Turn On".

Steps to remotely unlock from app

The lock must be connected to a gateway.

  1. Select a lock from the main menu, and confirm remote unlock is on.
  2. Tap on the small blue icon, then tap "Remote Unlock" when the confirmation prompt appears. The large blue lock icon is used for opening with eKeys and requires the user to be connected to the lock via Bluetooth.
  3. If remote unlock is successful, you will see a confirmation window.

Steps to remotely unlock from web system

If Remote Unlock is already enabled for a lock, you will see a WiFi icon directly under the lock name.

  1. Click on the lock you want to remotely unlock.
  2. On the lock page, click on "Remote", then click on "Remote Unlock".

Cards (RFID Keys)

Cards (aka RFID keys) are a more traditional type of access method.

Adding cards

Cards can be added into the system using a lock or by using a card reader.

1. Add a card using a lock

  • This requires being next to the hardware with Bluetooth enabled on your mobile device.
  • This process is done using the TTLock iOS or Android App only.
  • Any lock can be used to add a card into the system. Once added, it can be synced to the web system and assigned to other locks.

Steps to add a card using a lock

  1. From the home screen, select a lock.
  2. Tap "Cards".
  3. Choose if you want the card to be "Permanent", "Timed", or "Recurring". If "Timed" or "Recurring", set the validity period.
  4. Assign a name and tap next.
  5. The lock will ask you to present the card.
  6. Present the card against the front of the lock.

To program this card onto other locks after adding via a lock:

  1. Log into the TTLock Web Application.
  2. Click "Issue Card".
  3. Click "Sync Card" and the card you added to the lock will appear in the list.
  4. Click "Issue" next to the card and then choose the locks that you want that card to be programmed to and click "OK".

2. Add a card using a card reader

  • This is a TTLock Web Console only process.
  • The lock must be connected to a gateway.
  • The card encoder must be plugged into a computer logged into the TTLock Web Console.

Steps to add a card using a card reader

  1. From the home screen, click "Issue Card".
  2. Choose "Issue Card" from the menu bar.
  3. Select one or more locks you want the card to be programmed to.
  4. Name the card.
  5. Choose if you want the card to be "Permanent", "Timed", or "Recurring". If "Timed" or "Recurring", set the validity period.
  6. Click "Read Card".
  7. Tap the card on the card reader.

You can use the "Search" function by tapping the card against the card reader.

Delete a card

The lock must be connected to a gateway.

  1. From the home screen, click "Issue Card".
  2. Locate the card you wish to delete, then click "Door access".
  3. Select the locks from which you want to remove access (select the top box to remove from all locks).
  4. Click "Delete access", then "OK".
  5. Confirm that access has been removed from each lock.
  6. Click "OK" to complete.

eKeys (Digital Keys)

eKeys are digital keys.

eKey Types

  • Bluetooth eKeys: Use BLE technology which requires users to be in close proximity to the lock.
  • Remote Unlock eKeys: Allow users to open door locks from their mobile device using the internet. Users can open doors from anywhere, provided their mobile device has internet connectivity.

Types of eKeys include: Timed, One-Time, Permanent, or Recurring.

Adding eKeys

Once users are granted access to an eKey, they will be prompted to create a TTLock account and access the key from their mobile devices.

1. Steps to generate eKeys using app

  1. Select a lock from the main menu, then tap "eKeys", then tap "Send eKey".
  2. Choose from Timed, One-Time, Permanent, or Recurring eKey types and enter the required information. At this step, you can choose whether to also grant the user a "Remote Unlocking" key.

Once generated, the eKey can optionally be shared via email or directly from the app using a pre-set template.

2. Steps to generate eKeys using web system

  • This is a paid feature.
  1. Select a lock from the main menu, then tap "eKeys", then tap "Send eKey".
  2. Choose from Timed, One-Time, Permanent, or Recurring eKey types and enter the required information and tap "OK". Here, you can also select whether to grant the user "Remote Unlock" and/or "Authorized Administrator" permissions to the lock.

Tap "Send Email" to send the eKey to the recipient.

Fingerprints

Fingerprints are another access method.

Adding Fingerprints

  • This requires being next to the hardware with Bluetooth enabled on your mobile device.
  • Fingerprints must be programmed at each individual lock.

Types include: Timed, Permanent, or Recurring.

Steps to add fingerprints

  1. Select a lock from the main menu, then tap "Fingerprints", then tap "+ Add Fingerprint".
  2. Choose from Timed, Permanent, or Recurring fingerprint types, enter the required information, and tap "Next".
  3. Read the onscreen prompt, and tap "Start". Place your finger on the fingerprint sensor on your lock, and complete this 4 times as described by the audio commands.
  4. Once complete, you will see your Fingerprint Key listed on the Fingerprints Page.
