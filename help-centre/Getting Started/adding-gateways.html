Adding Gateways

Step 3 of the Getting Started Guide is to Add Gateways. To manage locks remotely, they must be paired to a gateway.

Important notes before adding gateways

Network Requirements

  • G2 (WiFi) gateways require a network that supports a 2.4GHz frequency.
  • G3 (Cable) or G3P (PoE) gateways require configuring firewall settings to permit connection through port 2999.
  • G4 (4G) gateways require obtaining a compatible SIM card and service provider.

Check Gateway Connection Strength

Tap “Nearby Locks” to bring up a list of locks that are currently connected to the gateway. From here you can do the following:

  1. View the individual locks that are connected to the gateway
  2. View the connection strength between each lock and the gateway

Each lock can be connected to multiple gateways. A lock that is showing a “Weak” connection to one gateway, may be showing a “Strong” connection to another.

  • This process is done using the TTLock iOS or Android App only.
  • You must be next to the hardware with Bluetooth enabled on your mobile device.
  • You need to confirm network requirements based on the gateway model.
  • Ensure that each lock is within a 6m radius of a gateway. Gateways should generally be installed as close to locks as possible, with a recommended maximum range of 20-30 feet, but this could be impacted by obstructions. Additional TTLock gateways can always be added after installation.

Steps to add a gateway

  1. Log into the app and tap the Menu icon, select "Gateway", and then tap the "+" icon on the top right corner.
  2. Tap on the correct gateway model (indicated on the label on the back).
  3. Physically connect the gateway to the power source (auxiliary or PoE).
  4. Once the gateway's LED is flashing red and blue, tap "Next". The gateway will time out if not connected within 60 seconds; to reset, unplug and plug it back in. It's best to add one gateway at a time.
  5. Tap the "+" symbol beside the gateway.
  6. For G2 WiFi gateways, you are required to select a WiFi network and enter the credentials. For G3 or G3P gateways, the device will automatically connect to the network. A flashing blue light means the gateway has connected to the network.
  7. Once the gateway is connected, it will automatically connect with nearby locks. You can check gateway to lock connection strengths by viewing "Nearby Locks" in the detailed view.
