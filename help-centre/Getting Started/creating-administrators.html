Creating Administrators or Transferring Ownership

Step 4 of the Getting Started Guide is an optional step: Set Up Administrators or Transfer Ownership.

Administrator Overview

  • When a new lock is added to an account, that account becomes the lock's administrator, having full control over lock settings and users.
  • Each lock has only one administrator.
  • An administrator can transfer a lock's administrator status to another account. This can only be performed by the lock administrator. For example, an installer might transfer status to their client after installation.

Transferring Locks

Follow the steps for transferring locks.

  • If a lock is transferred to another TTLock account, that account must have gateways added, or must be transferred gateways for the lock to be connected to the internet.

Steps to transfer a lock

This can only be performed by the lock administrator.

  1. From the main lock page, tap on the menu icon, and then tap "Settings".
  2. Tap "Transfer Lock(s)", select locks you want to transfer to another TTLock account and tap "Next".
  3. Enter a valid phone number or email registered to a TTLock account, then click "Next". You will then be prompted to confirm the transfer.

Once transferred, you will no longer have access to the lock. To regain access, the new administrator must add you as an authorized administrator.

Authorized Administrators

  • The lock administrator can assign any other registered accounts Authorized Administrator status.
  • Each lock can have multiple authorized administrators.
  • Authorized admin rights can be granted either temporarily or permanently.
  • They can issue credentials like passcodes or eKeys to other users.
  • They do not have the ability to add other authorized administrators to the lock. For example, a tenant can be granted authorized admin access to manage their own users while the building manager retains control.
  • Permissions are displayed below each lock on the main menu. Locks labeled "Permanent/Admin" indicate that you are the administrator and can add authorized administrators.

Steps to bulk add Authorized Administrators to a lock

This is a TTLock Web Console only process and can only be performed by the lock administrator.

  1. From the main lock page, click on the "eKey" icon for one of the locks you want to add an Authorized Administrator to.
  2. If you want to add this Authorized Administrator to other locks too, click on the "+" icon and select all the locks.
  3. Input the "Recipient Email", "Name", and select the "Type" of access. *The recipient email must already be registered with a TTLock account.
  4. Click the "Authorized Administrator" toggle, along with "Remote Unlock" if applicable.
  5. The recipient TTLock account will be notified, and will see all locks when logging into their account.

Steps to add an Authorized Administrator to a lock from the TTLock Mobile App

This can only be performed by the lock administrator.

  1. Select the lock from the main lock page then tap "Authorized Admin". If this icon is not present, you are not the administrator of the lock.
  2. Enter a phone number or email that is already registered to a TTLock account, then tap "Send".

Toggle "Manage their own users only" to prevent them from adding or deleting users other than those they have created. If toggled off, this user can delete users that others have created.

Transferring Gateways

You can also transfer gateways to another TTLock account. This can only be performed by the lock administrator.

Steps to transfer a gateway

  1. Log into the app, tap on the menu icon, and then select "Settings".
  2. Select "Transfer Gateway".
  3. Select one or more gateways you want to transfer to another TTLock account, then tap "Next".
  4. Enter a valid phone number or email registered to a TTLock account, then click "Next". You will then be prompted to confirm the transfer.

Once transferred, you will no longer have access to the gateway.
