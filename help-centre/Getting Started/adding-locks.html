Adding Locks

Step 2 of the Getting Started Guide is to Add Locks. Before adding a lock, it is recommended to follow the installation materials to install the lock on the door. A lock must be added to an account before it can be used.

Important points for adding locks

  • This process is done using the TTLock iOS or Android App only.
  • You must be next to the hardware with Bluetooth enabled on your mobile device.
  • You cannot add a lock that is already associated with another account. If the lock is associated with another account, you must either delete it from that account, transfer it from that account, or reset the lock.

Steps to add a lock

  1. Tap the Menu icon on the main lock page, tap "+ Add Lock", and then tap "Door Lock".
  2. Press any button on the lock keypad to activate pairing mode. You'll hear "Please add a Bluetooth administrator". Select the lock with the "+" icon from the list of nearby locks that appears. New locks will appear in blue; locks that have already been added to an account will appear in grey.
  3. Assign a name to the lock. For certain lock types, you'll also need to specify the door's opening direction.

After adding the lock, it's a best practice to **initialize lock settings for each lock**. This includes updating firmware, turning on remote unlock, and adjusting lock sound. These initial settings must be configured directly at each lock while you are next to the hardware with Bluetooth enabled.
