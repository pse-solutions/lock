Adding Authorized Administrators (bulk and single)

Authorized Administrators can be added to locks by the lock administrator. This can be done either in bulk, via the TTLock Web Console or individually, via the TTLock Mobile App.

Authorized Administrators can manage their own users, but cannot add or delete other Authorized Administrators.

Steps to Bulk Add Authorized Administrators to a Lock (TTLock Web Console only)

Can only be performed by the lock administrator

  1. From the main lock page, click on the "eKey" icon for one of the locks you want to add an Authorized Administrator to
  2. If you want to add this Authorized Administrator to other locks too, click on the "+" icon and select all the locks
  3. Input the "Recipient Email", "Name", and select the "Type" of access

*The recipient email must already be registered with a TTLock account. If it is not registered, request the recipient follow Account Registration steps before proceeding.

  1. Click the "Authorized Administrator" toggle, along with "Remote Unlock" if applicable.
  2. The recipient TTLock account will be notified, and will see all locks when logging into their account

Steps to Add an Authorized Administrator to a lock (from the TTLock Mobile App)

  1. Select the lock from the main lock page then tap "Authorized Admin". If this icon is not present, you are not the administrator of the lock.

Permissions are displayed below each lock on the main menu. Locks labeled "Permanent/Admin" indicate that you are the administrator and can add authorized administrators.

  1. Enter a phone number or email that is already registered to a TTLock account, then tap "Send"

Toggle "Manage their own users only" to prevent them from adding or deleting users other than those they have created. If toggled off, this user can delete users that others have created.
