Remote Unlock

Steps to turn on remote unlock

  1. Determine if remote unlock is already on.

If remote unlock is already turned on, you will see the words "Remote Unlock" in grey, directly under the lock name; OR you can see the blue network icon once a lock is selected.

  1. If remote unlock is not already turned on, select the lock that you want to enable remote unlock, then tap on "Settings"
  1. Tap on "Remote Unlock", then Tap the button to "Turn On".
